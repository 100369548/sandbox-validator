# 通讯沙盘填写规范（修订版 v3.1）

> 基于实际数据样本分析，适用于「通讯通路门店」「通讯客户沙盘」两个工作表  
> 适用范围：海南省（西区 / 南区 / 东区 / 海口），覆盖 19 个县/区

## 通用规则

### 数字字段
- 所有销量/容量/人数均填写**整数**，不带单位
- 单位已标注在表头（台/个/人/㎡）
- **不含分销和团单**（已标注字段）
- 取值范围：`0 ~ 999999`
- **所有门店**均需填写（即使为0也要显式填写）

### 枚举字段
- **严格按照给定枚举值填写，不可自创新词**
- 枚举值区分大小写（如 `TOP368` 不能写成 `top368`）
- 不要在 enum 值前后加空格

### 勾稽关系
1. **价位段 ⇄ 品牌**：分价位段销量之和 = 分品牌销量之和
   - 列20 + 21 + 22 = 列23 + 24 + 25 + 26 + 27 + 28
2. ** TOP位唯一**：每个品牌的 TOP1、TOP2、TOP3 不可重复
3. **促销员构成**：自有店员数 + 厂家促销员数 = 促销员总数（非强制，但建议满足）

### 编码引用
- **地理编码**（列8）：必须存在于 `地理信息` sheet
- **商圈编码**（列9）：必须存在于 `商圈` sheet
- **门店编码**（列6）：优先用荣耀系统编码，其次沙盘编码（`6`开头）
- **渠道编码**（列18）：PSI 系统 Channel ID，纯数字

### 命名规范
- 门店名称不含"市""区"字样
- 客户名称使用全称，**不可为空**
- 门店地址精确到街道/村委会级别

---

## 数据质量检查清单

- [ ] 位置信息完整（省 → 市 → 区县 → 乡镇 四级）
- [ ] 门店编码与规划进度一致（已存在 → 有编码，无规划 → 填"无规划"）
- [ ] 价位段求和 = 品牌求和
- [ ] TOP位不重复
- [ ] 客户名称不为空
- [ ] 枚举值在允许范围内
- [ ] 手机号格式正确（11位）
- [ ] 地理编码 / 商圈编码存在于对应 Sheet
