## 七、如何使用

### 1. 校验渠道编码
```
在 data_medium.xlsx 中搜索沙盘中的渠道编码：
如果存在 → 编码有效
如果不存在 → 编码可能无效或不在系统内
```

### 2. 校验地理编码
```
在 template.xlsx 中搜索沙盘中的地理编码：
如果存在 → 编码有效
如果不存在 → 需要修正为正确的系统编码
```

### 3. 校验客户名称
```
在 data_medium.xlsx 中搜索沙盘中的客户名称：
- 如果找到 O→A映射，使用 A名作为标准名称
- 如果在客户→渠道编码表中找到，确认渠道编码一致性
```

### 4. 校验商圈编码
```
检查沙盘中的商圈编码是否以 CNSQCN 开头：
- 是 → 符合编码规范
- 否 → 需要修正
在 data_small.xlsx 中验证有效性
```
